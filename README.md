# E-Commerce Website
This is a full-stack E-commerce web application that allows users to browse products, add items to cart, and manage orders. The project consists of a frontend, backend, and admin panel.

## Features

- User authentication
- Product listing
- Add to cart functionality
- Admin panel to manage products
- Backend API for product management

## Tech Stack

Frontend:
- React.js
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB

## Project Structure

E-commerce/
│
├── frontend     # React frontend
├── backend      # Node/Express backend
├── admin        # Admin dashboard

## Installation

Clone the repository

git clone https://github.com/Prajwal323-lang/ecommerce-website

Install dependencies

cd frontend
npm install

cd backend
npm install

Run the project

npm start
